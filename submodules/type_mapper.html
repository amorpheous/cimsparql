property generals¶

property, sparql & type are required

Sparql values should be like: http://iec.ch/TC57/2010/CIM-schema-cim15#PerCent this is how type or DataType usually looks like for each data point in the converted query result from SPARQLWrapper.

type can be anything as long as it is represented in the python_type_map.

Type

For sparql-types that are not sourced from objects of type rdf
