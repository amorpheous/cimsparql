Parameters

  • date (DateTime) – Date that is to be found a closest match to

  • dates (List[DateTime]) – A list of dates

Return type

cimsparql.parse_xml.get_cim_files(root_path, date_range)¶

For a given directory and a list/range of DateTimes, finds the paths to the files that are closest wrt. to the dates in date_range. Assumes date_range is sorted in ascending order

Parameters

  • root_path (Path) – Path to the root folder of a directory with sv & tp files

  • date_range (Iterable[DateTime]) – Iterable of pendulum DateRange

Returns: dictionary with date as key and another dictionary with path to the sv/tp files

Return type

Dict[DateTime, Dict[str, Path]]
