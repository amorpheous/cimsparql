Model¶

The cimsparql.model module contains the base class CimModel for both graphdb.GraphDBClient get_table as well as a set of predefined CIM queries.

The cimsparql.model module contains the base class CimModel for both graphdb.GraphDBClient with function get_table as well as a set of predefined CIM queries.

Query ac line segments

Parameters

  • region (Union[str, List[str], None]) – Limit to region

  • sub_region (bool) – Assume region is a sub_region

  • limit (Optional[int]) – return first ‘limit’ number of rows

  • connectivity (Optional[str]) – Include connectivity mrids as column name ‘{connectivity}_{1|2}’

  • rates (Tuple[str]) – include rates in output (available: “Normal”, “Warning”, “Overload”)

  • with_market (bool) – include marked connections in output

  • temperatures (Optional[List[int]]) – include temperature correction factors in output

  • dry_run (bool) – return string with sql query

Example

>>> from cimsparql.graphdb import GraphDBClient
>>> from cimsparql.url import service
>>> gdbc = GraphDBClient(service('LATEST'))
>>> gdbc.ac_lines(limit=10)
Return type

get_table(query, index=None, limit=None, map_data_types=True, custom_maps=None, columns=None)¶

Gets given table from the configured database.

Parameters

  • query (str) – to sparql server

  • index (Optional[str]) – column name to use as index

  • limit (Optional[int]) – limit number of resulting rows

  • map_data_types (bool) – gets datatypes from the configured graphdb & maps the types in the result to correct python types

  • custom_maps (Optional[Dict]) – dictionary of ‘sparql_datatype’: function to apply on columns with that type. Overwrites sparql map for the types specified.

  • columns (Optional[Dict]) – dictionary of ‘column_name’: function, uses pandas astype on the column, or applies function. Sparql map overwrites columns when available

Example

>>> from cimsparql.graphdb import GraphDBClient
>>> from cimsparql.url import service
>>> gdbc = GraphDBClient(service('LATEST'))
>>> query = 'select * where { ?subject ?predicate ?object }'
>>> gdbc.get_table(query, limit=10)
Return type

DataFrame

  • network_analysis (bool) – Include only network analysis enabled components

  • limit (Optional[int]) – return first ‘limit’ number of rows

  • dry_run (bool) – return string with sql query
